## 🛠️ API ARCHITECTURE

### **REST API Endpoints (22 Total)**

#### **Albums Management**
- `GET /api/memory-lane/albums` - List all albums
- `POST /api/memory-lane/albums` - Create new album
- `GET /api/memory-lane/albums/{id}` - Get specific album
- `PUT /api/memory-lane/albums/{id}` - Update album
- `DELETE /api/memory-lane/albums/{id}` - Delete album

#### **Media Upload**
- `POST /api/memory-lane/upload/photo` - Upload photos
- `POST /api/memory-lane/upload/video` - Upload videos  
- `POST /api/memory-lane/upload/voice` - Upload voice recordings

#### **Timeline Management**
- `GET /api/memory-lane/timeline` - Get timeline events
- `POST /api/memory-lane/timeline` - Add timeline event
- `PUT /api/memory-lane/timeline/{id}` - Update event

#### **Music & Stories**
- `GET /api/memory-lane/music/playlists` - Get music playlists
- `POST /api/memory-lane/music/playlists` - Create playlist
- `GET /api/memory-lane/stories` - Get captured stories
- `POST /api/memory-lane/stories` - Create new story

#### **Activities & Statistics**
- `POST /api/memory-lane/activities/start` - Start reminiscence activity
- `GET /api/memory-lane/stats` - Get usage statistics
