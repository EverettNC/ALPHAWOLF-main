# ALPHAWOLF: AI for Cognitive Support & Dementia/Alzheimer's Care

AlphaWolf is an ethical, accessible AI companion designed to preserve memories and dignity for those experiencing cognitive decline, dementia, or Alzheimer's. Built with empathy at its core, it empowers users and caregivers through intelligent, privacy-focused support—reconstructing lost moments, preventing wander risks, and providing gentle reassurance without eroding agency.

## Mission
No one should lose their memories—or their dignity. AlphaWolf combines structured logic with intuitive pattern recognition to create a safe, supportive environment that adapts to each user's unique needs. It's free forever, runs offline for weeks, and prioritizes human control at every step.

## Key Features
- **Memory Reconstruction**: Gently rebuilds timelines and recalls cherished moments through voice prompts and contextual cues, helping users anchor in the present.
- **Safety Zones**: Custom geofencing and wander prevention with soft alerts to caregivers, ensuring independence while providing peace of mind.
- **Voice Control**: Intuitive commands for easy interaction, with adaptive responses that honor emotional states.
- **Caregiver Support**: Transparent dashboards for monitoring and collaboration, always with user consent.

## Economics
Traditional memory care systems cost $13,000+. AlphaWolf is free—democratizing access to life-changing support without financial barriers.

## Installation (Local Setup)
1. Clone the repo: `git clone https://github.com/EverettNC/ALPHAWOLF.git`
2. Create venv: `python -m venv .venv; source .venv/bin/activate`
3. Install deps (add to requirements.txt if needed): `pip install langchain sympy fastapi`
4. Download spacy model: `python -m spacy download en_core_web_sm`
5. Run: `python app.py` (adapt for your main entry)
